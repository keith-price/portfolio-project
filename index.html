This React app was created to fetch data from NASA's APOD API. Each daily image is displayed with its title and description. It seemed a shame to then lose this incredible image when it is replaced the following day. So an archive page was created where users can browse previous APOD images and their related information. After fetching the data each day, the daily image along with its title and description are written to a database collection. That collection can be viewed as image cards on the archive page. Selecting an image then opens up the archive detail page with the image displayed in full along with its title and description.
